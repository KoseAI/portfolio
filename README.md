## 01. One-Stroke Path on Curved Surface（概念検証 / Python, Fusion 360）
- **課題**：曲面に対して一筆書きで連続パスを作り、複雑な造形物でも簡単に管理したい。
- **アプローチ**：輪郭抽出→曲面投影→Z補正→連続化ヒューリスティクス
- **結果**：サンプルSTLに対して点列CSV生成
- **学び**：前処理（法線/投影）と再現手順の明文化で結果の安定性が大きく向上
